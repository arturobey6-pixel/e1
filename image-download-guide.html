🖼️ CleanRide Garage — Canva Image Download Guide

Every image the website needs, matched to a ready-made Canva template. Download each design from Canva, save it with the exact file name shown, and drop it into the images/ folder of the theme. The HTML already points to these names — no code changes needed.

How to use (3 steps):
  1. Open the Canva link (Option 1 is the recommended pick; Options 2–4 are alternate styles). Edit it in Canva if you like.
  2. Download from Canva: Share → Download → choose JPG (or PNG where the file name ends in .png) → set the size shown in the table.
  3. Rename the downloaded file to the exact name in the "Save As" column and put it in the images/ folder (replace the placeholder that's already there).
⚠️ Dark placeholder images are already installed in images/ at the correct sizes, so the site works right now. Each one is labeled with its own file name — replace them with your Canva downloads whenever you're ready. For the favicon, download the logo as PNG at 512×512, then export/resize to 64×64 and save as favicon.png.
